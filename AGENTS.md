# Gentleman Guardian Angel Rules

## Arquitectura
- Las páginas deben ir en /app
- Los componentes reutilizables deben ir en /components
- Las rutas API deben estar en /app/api
- No se permite lógica de negocio en componentes UI

## Código
- No usar console.log en producción
- Usar async/await
- Evitar código duplicado
- Nombrar variables en inglés

## Seguridad
- No exponer API Keys en el código
- Usar variables de entorno
- Validar datos de entrada

## Calidad
- El código debe ser legible y modular
- Evitar funciones mayores a 50 líneas
